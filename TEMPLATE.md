# GitHub Portfolio Template

## Use This Template
1. Click "Use this template" on GitHub
2. Name your repo: `your-portfolio`
3. Clone and edit `script.js` line 51:
   ```javascript
   return params.get('user') || 'YOUR_USERNAME';
   ```
4. Push changes
5. Enable GitHub Pages
